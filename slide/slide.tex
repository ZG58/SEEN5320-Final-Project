\begin{frame}{Project Route}
\begin{block}{Goal}
Use successful gPROMS VPSA simulations to train a fast surrogate and optimize process KPIs.
\end{block}
\begin{itemize}
    \item Source data: \texttt{data/raw\_data.mat}.
    \item Successful simulations: 1708 rows from an original 2000-point LHS design.
    \item Inputs: FAD, FRP, FVU, TRP, TED, and TPR.
    \item Outputs: recovery, purity, productivity, and specific energy.
    \item Method: tune multiple ML models, select the best Pareto-aware surrogate, then run NSGA-II.
\end{itemize}
\end{frame}

\begin{frame}{Literature Basis}
\begin{columns}[T,onlytextwidth]

\begin{itemize}

    \item Their ANN surrogate enabled multi-objective optimization without repeatedly solving the full PDAE model.

\begin{frame}{NSGA-II Setup}
\begin{columns}[T,onlytextwidth]
\begin{column}{0.50\textwidth}
\small
\begin{block}{Problem 1}
\[
\max(\mathrm{purity},\mathrm{recovery})
\]
\end{block}
\begin{block}{Problem 2}
\[
\max \mathrm{productivity},\quad \min \mathrm{energy}
\]
\[
\mathrm{purity}\ge0.70,\quad \mathrm{recovery}\ge0.90
\]
\end{block}
\end{column}
\begin{column}{0.47\textwidth}
\small
\begin{itemize}
    \item Population size: 160.
    \item Generations: 250.
    \item Operators: simulated binary crossover and polynomial mutation.
    \item Ranking: constrained nondominated sorting plus crowding distance.
\end{itemize}
\end{column}
\end{columns}
\end{frame}

\begin{frame}{Conclusions}
\begin{itemize}
    \item The MAT file was converted into a clean scalar table with 1708 successful VPSA simulations.
    \item Eight surrogate families were tuned and compared with global and Pareto-focused metrics.
    \item RBF SVR was selected as the best optimization surrogate.
    \item NSGA-II produced interpretable Pareto fronts for both objective formulations.
    \item The workflow now connects data conversion, surrogate learning, optimization, figures, report, and slides.
\end{itemize}
